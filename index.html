1. Introduction

Welcome to Waferlee("we," "our," or "us") . We are committed to protecting your privacy while providing you with the best coupon-finding and price comparison experience for UAE online shopping.

This Privacy Policy explains how our Chrome extension collects, uses, discloses, and safeguards your information when you use Waferlee.

üìå By installing and using Waferlee, you agree to the collection and use of information in accordance with this policy.

If you do not agree with our policies and practices, please do not install or use the extension.

2. Information We Collect

2.1 User Actions

  • Product searches you initiate via the extension
  • Coupon codes you copy
  • Links you click to visit stores
  • Modal interactions (minimize, close, open)

2.2 Automatically Collected Information

Browsing Data (Limited Scope):

  • Current Tab URL: Only collected when you actively visit a partner shopping website to check for available coupons
  • Selected Text: Only when you right-click and choose "Find Cheaper Deals"
  • Domain Information: To identify if you're on a partner store and show relevant coupons
  • Active Tab Information: To update the extension badge counter

Usage Data:

  • Timestamps of searches and coupon copies
  • Number of price comparisons performed
  • Search success/failure rates (anonymized)
  • Extension interaction events (tab switches, button clicks)

Technical Data:

  • Browser version and type (Chrome)
  • Extension version number
  • Error logs and crash reports (anonymized)
  • API response times (for performance monitoring)

2.3 Information We Do NOT Collect

  • Your browsing history beyond active shopping sessions
  • Personal identification information (name, email, phone)
  • Payment or credit card information
  • Passwords or login credentials
  • Full page content or form data
  • Information from non-shopping websites
  • Cross-site tracking or fingerprinting data

3. How We Use Your Information

3.1 Primary Purposes

To Provide Core Functionality:

  • Automatically detect when you visit partner shopping websites
  • Search and display available coupon codes from our database and DCM Network API
  • Compare product prices across UAE online stores using Google Shopping data
  • Show available discounts and savings opportunities
  • Display personalized notifications based on your preferences
  • Maintain your recent price comparison history (stored locally, last 20 searches)
  • Update badge counter showing available coupon count for current website

To Improve User Experience:

  • Remember your settings and preferences across sessions
  • Avoid showing dismissed coupon notifications repeatedly (respects "closed" state per website)
  • Prioritize coupons from the website you're currently visiting (shown first with green badge)
  • Cache API results to reduce loading times (1 hour for coupons, 30 minutes for prices)
  • Paginate coupons (10 per page) for easier browsing
  • Filter duplicate coupons from multiple sources (DCM Network + Google Sheets)

To Maintain and Improve the Service:

  • Analyze aggregated usage patterns to identify popular features
  • Debug technical issues and application crashes
  • Monitor API performance and reliability (Google Sheets, SerpAPI, DCM Network)
  • Optimize search algorithms for better coupon matching
  • Rate limiting to prevent service abuse (5 SerpAPI requests/minute, 10 coupon checks/minute)

3.2 Legal Bases for Processing (GDPR/UAE DPA Compliance)

We process your information based on:

  • Consent: You provide explicit consent by installing and using the extension
  • Legitimate Interest: To operate and improve our service
  • Contractual Necessity: To provide the features you've requested

5. Data Storage and Security

5.1 Where Your Data is Stored

Local Storage (On Your Device):

  • Recent price comparison history (last 20 searches)
  • Dismissed coupon notification states
  • Duration: Until you uninstall the extension or clear data

Our Servers (Google Cloud Infrastructure - UAE/EU Regions):

  • Aggregated, anonymized usage statistics
  • Coupon database (no user-specific data)
  • Duration: Anonymized analytics retained for 90 days

5.2 Security Measures

Technical Safeguards:

  • ‚úÖ All API communications use HTTPS/TLS encryption
  • ‚úÖ No passwords or credentials stored
  • ‚úÖ Content Security Policy (CSP) to prevent code injection
  • ‚úÖ Sandboxed iframes for third-party widgets
  • ‚úÖ Rate limiting to prevent abuse
  • ‚úÖ Regular security audits and updates

Access Controls:

  • Only essential team members can access aggregated data
  • No single person has access to decrypt individual user data
  • Multi-factor authentication required for all admin access

5.3 Data Breach Protocol

  2. Report to UAE Data Office and relevant authorities
  3. Provide guidance on protective measures
  4. Conduct thorough investigation and implement fixes

Waferlee requests the following permissions from Chrome. Here's why each one is needed:

Permission Why We Need It What We Do With It
storage Save your preferences locally Store settings (region, currency, notifications), recent comparisons (last 20), modal states per website. All stored on YOUR device only.

activeTab Read current website URL Detect which shopping website you're on to show relevant coupons. Only when tab is active.

(https://*/*)		 Inject coupon notifications Display coupon modal on shopping websites you visit. Required to show coupons on any e-commerce site. Does NOT access banking, email, or sensitive sites.
